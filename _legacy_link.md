# 索引工具已迁移

**新位置**：`50-项目/工具/索引/`（通用，跨项目复用）

## 变化

| 旧路径 | 新路径 | 状态 |
|---|---|---|
| `50-项目/主网自动化竞赛/工具/build_index.py` | `50-项目/工具/索引/build_index.py` | 旧脚本**保留**兼容，新增 `--root` / `--ocr` 参数只在新位置 |
| —（无） | `50-项目/工具/索引/search.py` | 新增：全文搜索（cmd+shift+f 等价物） |
| —（无） | `50-项目/工具/索引/ocr_pdf.py` | 新增：OCR 兜底（cmd+F 等价物） |

## 迁移起因（2026-05-18）

回答综合智能告警题目时，`资料库索引.json` 里只有 desc 预览 + 高频关键词，搜不到 PDF 全文里的 `isw_fault_client` / `fes_sim` 这类专有词。根因：`.extracted-text/` 下的几百万字全文从未被纳入搜索路径。

新流程：以后任何项目搜资料库都走 `50-项目/工具/索引/search.py`，不要只查 JSON。

详细使用见 `50-项目/工具/索引/README.md` 和 `.claude/skills/索引.md`。
